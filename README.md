# 🚀 SkillForge UI

A modern, interactive EdTech landing page built using HTML, CSS, and JavaScript.  
This project focuses on delivering a premium UI/UX experience with advanced animations, smooth interactions, and responsive design.

---

## 🌐 Live Demo

🔗 https://l24cs200.github.io/SkillForge-UI/

---

## 📌 Project Overview

SkillForge is a visually immersive frontend project designed to demonstrate modern web design principles including:

- Clean UI structure
- Responsive layout
- Advanced CSS animations
- Micro-interactions
- Smooth scrolling navigation
- Interactive pricing and course sections
- Animated splash screen
- Glassmorphism and gradient effects

The goal of this project was to create a premium, engaging learning platform interface with strong attention to user experience and performance.

## 🛠 Tech Stack
